Detailed Description

template<class ExpectedResult>
class unittest::Test< ExpectedResult >

The Test template class is an abstracted, simple class to display the results of unit tests. Each test can be named, given an expected value, and then a true value: test.name("MyClass::myMethod()").expect("Foo").test( myClass.myMethod() ); This will generate output to stderr shwoing the expected value, actual value, and the pass/fail status.

Parameters:
ExpectedResult the type of results expected for this test. It is common for a test suite to instantiate a number of Test classes for different datatypes.
